SQL project for managing student results
# Student Result Management System (SQL)

This project implements a simple relational database using MySQL to manage
students, subjects, and their marks.

## Tables
- students
- subjects
- marks

## Features
- Store student and subject details
- Record marks for each subject
- Retrieve student-wise results
- Calculate average marks
- Identify top-performing students

## Files
- schema.sql – database table creation
- insert_data.sql – sample data insertion
- queries.sql – SQL queries for analysis

## Tools Used
- MySQL
- MySQL Workbench
